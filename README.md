# Open Obsidian

A simple redirector for Obsidian URI schemes.

## Usage

Append `?url=` followed by an encoded Obsidian URI to the base URL.

Example:
`https://stefanhoth.github.com/open-obsidian?url=obsidian://open?vault=openclaw&file=020%20-%20%F0%9F%92%A1%20Knowledge%2F001%20-%20Rezepte%2F008%20-%20Kartoffel-Linsen-Eintopf`
